Modules

A toolkit in four acts

From discovery to inference, everything stays on your machine.

Scout

Search Hugging Face for coding models. Fit-check estimates RAM usage against your machine before you download a single byte.

interference scout "qwen coder"
RF-SCOUT

Manager

Download models exclusively to your external SSD. Golden-rule enforcement: internal disk is off-limits. Resume and integrity built-in.

interference pull mlx-community/Qwen2.5-Coder-14B-4bit
RF-MGR

Orchestrator

OpenAI-compatible server on localhost. MLX-powered inference with hot-swap multi-model — fast model for completions, strong model for reasoning.

interference serve
RF-ORK

Glue

One command wires opencode to your local orchestrator. Zero configuration — start coding with local models immediately.

interference opencode-config
RF-OC

Why

Not just another model runner

Ollama and LM Studio solve part of the problem. Interference solves the whole pipeline — from discovery to the coding agent.

✓
Scout with fit-check Know if a model fits your RAM before downloading. No more guessing, no more wasted bandwidth.
✓
Golden rule: external SSD only Models are large. Your internal disk is small. Interference refuses to touch it — enforced at every layer.
✓
opencode integration One command wires your local models to a Claude Code–style coding agent. Not just an API — a complete workflow.
✓
Hot-swap multi-model Fast model for completions, strong model for reasoning. Switch on the fly without restarting the server.

Experience

Terminal to agent in four commands

1
interference scout "qwen coder"
See what fits your machine. No download yet.
2
interference pull <model>
Download lands on your external SSD. Internal disk: untouched.
3

4

That's it. No config files, no port forwarding, no cloud sign-up. From zero to local coding agent in under a minute.

Ready to run local?

Install the toolkit, scout for models, and start coding with local LLMs — all from the terminal.

pip install interference[serve]
interference doctor
interference scout "qwen coder"
interference pull <model>
interference serve
git clone https://github.com/.../interference
cd interference
uv sync --extra serve
uv run interference doctor
uv run interference scout "qwen coder"
